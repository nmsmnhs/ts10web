# ts10web
Brief Project Summary and Description
Our project for Fulbright STEM Mentorship Program is an AI-powered web application that helps Vietnamese students prepare for the National High School Entrance Exam. By offering a wide range of English practice questions alongside detailed AI-generated explanations, we aim to make test prep more convenient and effective, and help students across the country feel more confident about this important exam.

Project Final Results

Achievements and overall results
By the end of the project, we successfully developed the web app as planned. It currently includes:

A dropdown menu with 8 question categories for users to choose from

Real-time answer checking

Detailed and accurate AI-generated explanations

A clean, user-friendly interface

Overall, the app meets all the main objectives and is ready for deployment.

Community involvement
We invited a few students to test the app and share their thoughts. Many said it was convenient and helpful—not just for 9th graders, but for students in general. They liked that it’s free, ad-free, and accessible for those who can’t afford extra classes.
That said, there were some points of feedback we’ll keep in mind:

Only getting one question at a time was a bit frustrating

A scoring system or tracking correct answers would improve the experience
